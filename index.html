Модальное окно

Это пример модального окна. Оно открывается по кнопке и закрывается по крестику.

Аккордеон (accordion) — это вертикальный набор раскрывающихся блоков. При клике на заголовок открывается панель с содержимым. Обычно реализуется на JavaScript, но здесь мы используем только CSS и псевдокласс :checked.

Скрытый чекбокс переключается при клике на <label>. Когда чекбокс отмечен (:checked), соседний блок с содержимым получает свойство max-height с конкретным значением, и он плавно появляется. Так достигается анимация без единой строчки JS.

Этот метод легковесный, не требует загрузки скриптов, работает даже при отключённом JavaScript. Подходит для FAQ, справочных разделов, мобильных меню. Главное — правильно подобрать значение max-height, чтобы анимация была плавной.
